# vant3 + vue3 + ts开发模板
## 基于vue3的一款自用的快速开发模板，已经集成了 axios, 登录组件，轮播组件，以及视频播放器组件。
<br>

### 项目的开发及构建
```javascript
// 1. 开发：怎么让项目跑起来
npm install && npm run dev

// 2. 构建：就是打包啦，生产环境下用的。
npm run build
```
